== Description ==

<strong>Podamibe Facebook Widget</strong> is a <strong>Free WordPress plugin</strong> that displays Facebook feeds on your pages in your website. Facebook feed is very easy to use and in few minutes you can show any public facebook page feed on your website. Each social network can be signed into individually and there are several options to customize the displayed feed.<br />

<strong>Free Features:</strong>
<ul>
<li>Main icon to display on front</li>
<li>Main title to display on front</li>
<li>Set the width of feed widget for display</li>
<li>Set the height of feed widget for display</li>
<li>Small Header Option</li>
<li>Cover Image Option</li>
<li>Display the liked users</li>
<li>Select the option either you want to display the users images or not</li>
<li>Set the border width of feed on front</li>
<li>Tabs to section.</li>
</ul>

== Installation ==

1. Upload the plugin 'podamibe-facebook' to the `/wp-content/plugin s/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance->Widgets->Podamibe Facebook Feeds
4. Drag the widget to your desired sidebar and fill the fields and save


== Frequently Asked Questions ==

= How to insert the main icon? =
Click the link provided below the field. Then enter name like 'facebook' only for facebook icon.

= What is main title? =
This is the title to be displayed nearby the icon.

= What is facebook page url ? =
This is field to insert your facebook page url from your facebook.

= What is facebook App id? =
This is the application id generated on facebook of your facebook page.

=  What is width and height? =
The custom height and width for widget to display on frontend.

= What is small header option? =
This is the option to display the header either smaller or larger of facebook page.

= What is cover image option? =
This is the option to display the facebook page cover image.

= What is Liked User? =
This is the option to display the users image who liked your facebook page.
